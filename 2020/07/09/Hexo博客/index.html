发表于 2020-07-09 | 分类于 Hexo

Hexo安装

1、安装node.js

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。在Mac上可以通过安装包安装、二进制库安装和源码安装3种方式来进行Nodejs环境搭建。最简单是的使用安装包安装,首先进入nodejs下载网址http://nodejs.cn/download/，点击“macOS 安装包”进行下载。下载后双击打开安装包，和正常安装软件一样安装完成即可。如果安装成功后还是提示没有找到模块，那可能是没有配置node.js的环境变量。具体可参考简书:https://www.jianshu.com/p/66b86ad6b48a

安装之后可以查node.js和npm包管理器的版本号来验证软件是否安装正确

2、安装hexo

使用包管理器npm来安装hexo，由于源是国外的,如果访问很慢，可以添加国内镜像再安装，如果遇到 Error: EACCES: permission denied, 命令前面请添加sudo
